# 2026年7月中国干支日历

一个可在 Windows 本地离线查看和打印的整月日历，包含公历、农历、年柱、月柱、日柱、节气和少量纪念日。

## Windows 使用方法

1. 保持整个 `2026年7月中国干支日历` 文件夹完整，不要单独移动其中的文件。
2. 双击 `index.html`。
3. 系统会使用默认浏览器打开日历，不需要安装软件，也不需要联网。

推荐使用当前版本的 Microsoft Edge、Google Chrome 或 Firefox。

## 打印或保存 PDF

1. 点击页面右上角的“打印 / 保存 PDF”。
2. 在打印窗口中选择打印机，或选择“另存为 PDF”。
3. 建议使用 A4、横向、默认缩放；如浏览器提供“背景图形”选项，可将其打开以保留周末底色。

## 已完成功能

- 2026 年 7 月完整自然周月历，星期一为第一列
- 公历日期、农历月份与日期
- 丙午年年柱
- 按节气切换的月柱（甲午月、乙未月）
- 每日天干地支日柱
- 小暑、大暑
- 建党节、七七事变纪念日
- 周末、今天、节气、纪念日和相邻月份日期的视觉区分
- 桌面端、窄屏和 A4 横向打印样式
- 历法库加载失败时的页面提示

## 数据方案

农历、干支和节气由本地固定版本 `lunar-javascript 1.7.7` 计算：

- 项目主页：`https://github.com/6tail/lunar-javascript`
- npm 包：`https://www.npmjs.com/package/lunar-javascript`
- 本地文件：`vendor/lunar.js`
- 上游许可：MIT，副本见 `vendor/LICENSE.lunar-javascript`

节日和纪念日使用 `calendar-core.js` 中的独立映射，便于继续补充。

## 数据准确性说明

本项目不手工写死整月农历或干支数据，关键字段统一由历法库生成，并对 2026 年 7 月 14 日、小暑和大暑等样本进行自动测试。不同历法软件可能因节气交接时刻、时区或算法版本产生边界差异；正式出版或民俗专业用途，建议再与权威历书核对。
